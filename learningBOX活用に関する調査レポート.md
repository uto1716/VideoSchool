# learningBOX活用に関する調査レポート

## 調査概要

本レポートは、女性限定動画編集スクールのLP（ランディングページ）制作にあたり、learningBOXの活用方法について調査した結果をまとめたものです。

---

## 調査結果の要約

調査の結果、**learningBOXの無料プランでは、外部に公開する独立したLP（ランディングページ）を作成する機能は提供されていない**ことが判明しました。

learningBOXは本質的に**eラーニングシステム（LMS: Learning Management System）**であり、その主な機能は以下の通りです。

| 機能カテゴリ | 内容 |
| :--- | :--- |
| 学習コンテンツ管理 | 動画、クイズ、Webページ形式の教材を作成・配信 |
| ユーザー管理 | 受講者の登録、グループ管理、権限設定 |
| 成績管理 | 学習進捗の追跡、成績のグラフ・表での閲覧 |
| AIアシスト | 学習支援のためのAI機能 |

---

## 選択肢の再整理

当初ご検討されていた2つの選択肢について、調査結果を踏まえて再整理します。

### 選択肢1：learningBOXでLPを作成する

**結論：現実的ではない**

learningBOXには「Webページ作成機能」がありますが、これは**ログイン後に閲覧する学習教材**として使用されるものであり、外部公開用のLPとしては機能しません。

デザインをカスタマイズするには「サイトカスタマイザー」機能が必要ですが、これは**カスタマイズオプション（初年度22万円）**の契約が必要です。さらに独自ドメインを使用するには**追加で年間16.5万円**が必要となります。

| 項目 | フリープラン | 有料オプション |
| :--- | :--- | :--- |
| LP作成機能 | なし | なし（LMSであるため） |
| デザインカスタマイズ | 不可 | 初年度22万円で可能 |
| 独自ドメイン | 不可 | 年間16.5万円で可能 |
| コピーライト非表示 | 不可 | 買い切り26.4万円で可能 |

### 選択肢2：LPは独自で作成し、CVした顧客をlearningBOXに流す

**結論：唯一の現実的な選択肢**

この方法が最も効果的です。具体的な運用フローは以下の通りです。

1. **独自LP**で集客・訴求を行う
2. LPから**無料相談会や資料請求**などのCVポイントを設置
3. CVした見込み客に対して**個別カウンセリング**を実施
4. 契約後、**learningBOXに受講者として登録**
5. learningBOX上で**学習コンテンツを提供・進捗管理**

## 推奨アプローチ

以上の調査結果を踏まえ、以下のアプローチを推奨します。

### LP制作について

独自でLPを制作することを推奨します。制作方法としては以下の選択肢があります。

| 方法 | メリット | デメリット | 概算コスト |
| :--- | :--- | :--- | :--- |
| **ノーコードツール（STUDIO, Wixなど）** | 自分で更新可能、比較的安価 | デザインの自由度に制限あり | 月額1,000〜3,000円程度 |
| **Web制作会社に依頼** | 高品質なデザイン、プロの知見 | コストが高い、更新に時間がかかる | 30〜100万円程度 |
| **本サンプルのようなコード実装** | 完全なカスタマイズ、高速表示 | 技術的知識が必要 | 開発費による |

### learningBOXの活用について

learningBOXは**学習管理システム**として活用することを推奨します。

フリープランでも以下の機能が利用可能です。

- 10アカウントまでの受講者管理
- 1GBまでのコンテンツ保存
- 動画・クイズ・Webページ形式の教材作成
- 成績管理・進捗追跡

受講者数が増えた場合は、スタータープラン（100アカウント/月額5,500円）への移行を検討してください。

## 添付：LPサンプル

本調査に合わせて、「独自LP」のサンプルを作成しました。このサンプルは、女性限定動画編集スクールを想定し、以下のセクションで構成されています。

1. **ヒーローセクション** - キャッチコピーとメインビジュアル
2. **悩み共感セクション** - ターゲットの課題を提示
3. **特徴セクション** - スクールの強みを紹介
4. **カリキュラムセクション** - 3ヶ月の学習内容
5. **卒業生の声セクション** - 実績と信頼性の証明
6. **料金プランセクション** - 2つのプランを提示
7. **受講の流れセクション** - 申し込みから開始までのステップ
8. **FAQセクション** - よくある質問への回答
9. **CTAセクション** - 最後の行動喚起

デザインは「Organic Minimalism（オーガニック・ミニマリズム）」を採用し、女性ターゲットに適した温かみのある色調と、十分な余白を活かしたレイアウトとしています。
