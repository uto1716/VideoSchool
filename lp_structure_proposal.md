# 女性限定動画編集スクールLP 構成案

## 調査結果のまとめ

はじめに、learningBOXの機能について調査した結果をご報告します。調査の結果、**learningBOXの無料プランでは、外部に公開する独立したLP（ランディングページ）を作成する機能は提供されていない**ことが判明しました。

- **learningBOXはeラーニングシステム**: あくまで学習管理システム（LMS）であり、ログイン後の学習環境を提供することが主目的です。
- **デザインの制約**: 外部向けのLPとして必要な、自由なデザインカスタマイズは有料オプション（初年度22万円〜）が必要となり、フリープランでは利用できません。

このため、当初ご検討されていた「learningBOXでLPを作成する」という選択肢は現実的ではなく、**「LPは独自で作成し、申し込みがあった顧客（CVユーザー）をlearningBOXに登録して学習を管理する」**という方法が唯一の現実的な選択肢となります。

つきましては、その「独自で作成するLP」をどのような内容にすべきか、具体的な構成案をご提案します。女性向けサービスという点を考慮し、共感と信頼を生むストーリーを重視した構成としています。

---

## LP構成案：7つのセクション

以下に、LPの具体的な構成要素を7つのセクションに分けてご提案します。

### 1. ファーストビュー：心をつかむ第一印象

ユーザーがページを開いて最初に目にする、最も重要なエリアです。ここで興味を惹きつけ、続きを読む動機を与える必要があります。

- **キャッチコピー**: 「未経験から、私らしい働き方を叶える。3ヶ月でプロになる、女性のための動画編集スクール」など、ターゲットの夢や目標に直接響く言葉を選びます。
- **メインビジュアル**: 実際に学ぶ女性たちの楽しそうな写真や、制作された動画のおしゃれなイメージ画像を配置し、理想の未来を視覚的に伝えます。
- **CTA（Call to Action）**: 「まずは無料相談会に参加する」「限定eBookをダウンロード」など、次の行動を促すボタンを分かりやすく設置します。

### 2. 共感・課題提起：あなたの悩み、分かります

ターゲットが抱えているであろう悩みや課題を具体的に提示し、「これは私のことだ」と感じてもらうセクションです。

- **悩みのリストアップ**: 「将来のためにスキルを身につけたいけど、何から始めればいいか分からない」「育児と両立できる仕事がしたい」「クリエイティブな仕事に憧れる」といった具体的な悩みを箇条書きで示します。
- **共感を呼ぶストーリー**: 編集部や講師の体験談などを交え、悩みに寄り添う姿勢を見せます。

### 3. 解決策の提示とベネフィット：未来はこう変わる

提示した課題に対して、このスクールがどのように役立つのかを明確に示します。単なる機能説明ではなく、受講することで得られる「未来（ベネフィット）」を語ることが重要です。

- **得られる未来像**: 「場所と時間に縛られない自由な働き方」「自分の作品で人を喜ばせるやりがい」「同世代の仲間との出会い」など、受講後の輝く未来を提示します。
- **Before/After**: 受講前の悩みと、受講後の変化を具体的に示すことで、変化を自分事として捉えてもらいます。

### 4. サービスの特徴：選ばれる理由

他のスクールではなく、なぜこのスクールが選ばれるのか。具体的な強みを分かりやすく伝えます。

| 特徴 | 具体的な内容 |
| :--- | :--- |
| **女性限定の安心環境** | 男性がいない環境で、気兼ねなく質問・相談ができるコミュニティ。 |
| **現役プロ講師による実践指導** | 現場で活躍するプロの動画クリエイターが、最新のスキルを直接指導。 |
| **個別最適化されたカリキュラム** | 一人ひとりの目標やレベルに合わせた、オーダーメイドの学習プラン。 |
| **卒業後も続くキャリアサポート** | ポートフォリオ制作のサポートから、案件獲得のノウハウまで提供。 |

### 5. お客様の声・実績：信頼の証

第三者の声は、信頼性を高める上で非常に効果的です。顔写真や実名（可能であれば）を掲載することで、信憑性が増します。

- **卒業生のインタビュー**: どのような経緯で入学し、どのようにスキルを習得し、現在どのように活躍しているかをストーリー形式で紹介します。
- **制作実績**: 卒業生が制作した動画作品を掲載し、スクールの質の高さを証明します。

### 6. 料金プランと受講の流れ：安心して申し込むために

料金体系は明確にし、申し込みから受講開始までの流れをステップで示すことで、ユーザーの不安を取り除きます。

- **料金プラン**: 「一括払い」「分割払い」など、複数の支払い方法を用意し、それぞれの料金を明記します。
- **受講開始までのステップ**: 「1. 無料相談会申し込み → 2. 個別カウンセリング → 3. ご契約 → 4. 学習スタート」のように、具体的な流れを図解します。

### 7. よくある質問とCTA：最後の一押し

最後の不安を解消し、行動を後押しするセクションです。

- **Q&A**: 「PCスキルに自信がありませんが大丈夫ですか？」「本当に未経験からプロになれますか？」といった、想定される質問への回答を予め用意しておきます。
- **最後のメッセージとCTA**: スクールへの想いを伝え、「あなたの挑戦を応援しています」といったメッセージと共に、再度申し込みボタンを設置します。
