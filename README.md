# VideoSchool - 女性向け動画編集スクールLP

25〜39歳の女性向け動画編集スクールのランディングページ

## 🎨 特徴

- **デザイン**: Warm & Approachable（ベージュ、ダスティピンク、アイボリー）
- **フレームワーク**: React + Vite + TypeScript
- **スタイリング**: Tailwind CSS
- **アニメーション**: Framer Motion
- **レスポンシブ**: スマホ・タブレット・PC完全対応

## 💻 ローカル開発

```bash
# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```
